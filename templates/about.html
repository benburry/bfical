About

BFiCal is an appengine application that generates an icalendar version of the BFI listings (currently only for the southbank) by screenscraping bfi.org.uk. The intention is that you can import this .ics file into something that does calendars properly. And iCal. snark.

The same data is used to generate this site, letting you navigate the listings by day/week, view listing by year/participant and save calendar reminders for individual showings.

One day the BFI might offer a datafeed. I can dream.

Why

I love the BFI, but I find their website incredibly frustrating to navigate. BFiCal is an attempt to make that process easier.

I'm keeping the number/frequency of data requests to a reasonably low level - the spider uses a custom user-agent and honours robots.txt if that were to ever become a problem.

Who
